Aprendendo HTML: Tema da aula 02 Listas

Hoje vamos aprender a como criar LISTAS usando html, lembrando dos elementos da aula passada.

Usando o elemento (ul) criamos um espaçamento. Veja no exemplo a seguir:

  • Cafe
  • Cerveja
  • Coca-cola
  • Agua
Usando o elemteno (li) criamos um espaçamento com uma *.

Para criar uma Lista Ordenada, use o elemento (ol) como no exemplo abaixo:

  1. Matematica
  2. Portugues
  3. Programacao Internet
  4. Algoritimo

Podemos tambem criar uma Lista Ordenada com Letras adicionando uma propriedade assim: (ol type="A")

  1. Matematica
  2. Portugues
  3. Programacao Internet
  4. Algoritimo

Ou tambem podemos escrever uma propriedade em numeros romanos com a propriedade (ol type="I">, veja abaixo o exemplo:

  1. Matematica
  2. Portugues
  3. Programacao Internet
  4. Algoritimo
